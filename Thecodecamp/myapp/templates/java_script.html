1. JS HOME

This guide provides a comprehensive overview of JavaScript, covering basic to advanced topics to help you become proficient in the language.

2. JS Introduction

JavaScript is a versatile scripting language used to create dynamic and interactive web content. It runs in web browsers and can also be used on the server side with Node.js.



    


        

            
3. JS Where To

            
JavaScript can be included directly in HTML documents between <script> tags or referenced externally using the <script src="filename.js"> tag.

        
<!DOCTYPE html>
<html>
<body>

<h2>JavaScript in Body</h2>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "My First JavaScript";
</script>

</body>
</html>

        

        


        

            
4. JS Output

            
Writing into an HTML element, using innerHTML.
                Writing into the HTML output using document.write().
                Writing into an alert box, using window.alert().
                Writing into the browser console, using console.log().
                Output JavaScript results using console.log() or by modifying the HTML content:

            
        <!DOCTYPE html>
        <html>
        <body>

        <h2>My First Web Page</h2>
        <p>My First Paragraph.</p>

        <p id="demo"></p>

        <script>
        document.getElementById("demo").innerHTML = 5 + 6;
        </script>

        </body>
        </html>

<html>
<body>

<h1>My First Web Page</h1>

document.write(5 + 6);

</body>

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<p id="result"></p>
<button type="button" onclick="document.getElementById('result').textContent = 5 + 6">Try it</button>

</body>
</html>
            

        


        

            
5. JS Statements

            
Statements are individual instructions that perform actions. In JavaScript, statements end with a semicolon ;.

            
<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Statements</h2>

<p>JavaScript statements are separated by semicolons.</p>

<p id="demo1"></p>

<script>
let a, b, c;
a = 5;
b = 6;
c = a + b;
document.getElementById("demo1").innerHTML = c;
</script>

</body>
</html>

            

        


        

            
6. JS Syntax

            
JavaScript syntax includes rules for writing statements and code blocks. Code blocks are enclosed in curly braces {}.

            
        if (condition) {
            // code block
        }
            

        


        

            
7. JS Comments

            
Comments are used to annotate code and are ignored by the JavaScript engine. Use // for single-line comments and /*...*/ for multi-line comments.

            
        // Single-line comment
        /*
         Multi-line comment
        */
            

        


        

            
8. JS Variables

            
Variables store values. Declare variables using let, const, or var:

            
        let name = "John";
        const age = 30;
        var city = "New York";
            

        


        

            
9. JS Let

            
let is used to declare variables that can be updated:

            
        let score = 10;
        score = 20; // Updated value
            

        


        

            
10. JS Const

            
const is used to declare variables that cannot be reassigned:

            
        const pi = 3.14;
        // pi = 3.14159; // Error: Assignment to constant variable
            

        


        

            
11. JS Operators

            
Operators are used to perform operations on variables and values. Common types include arithmetic, assignment, and comparison operators:

            
        let a = 5 + 3; // Addition
        let b = 5 - 2; // Subtraction
        let c = 5 * 2; // Multiplication
        let d = 10 / 2; // Division
        let isEqual = (a == b); // Comparison
            

        


        

            
12. JS Arithmetic

            
Arithmetic operators perform mathematical operations:

            
        let sum = 10 + 5; // Addition
        let product = 10 * 5; // Multiplication
            

        


        

            
13. JS Assignment

            
Assignment operators assign values to variables. The most common is =:

            
        let x = 5;
        x += 2; // x is now 7
            

        


        

            
14. JS Data Types

            
JavaScript supports various data types including strings, numbers, booleans, objects, and arrays:

            
        let str = "Hello";
        let num = 100;
        let isTrue = true;
        let obj = { name: "John", age: 30 };
        let arr = [1, 2, 3];
            

        


        

            
15. JS Functions

            
Functions are blocks of code designed to perform a task. They can be defined using the function keyword:

            
        function greet(name) {
            return "Hello, " + name;
        }
        console.log(greet("Alice")); // Hello, Alice
            

        


        

            
16. JS Objects

            
Objects store collections of data as key-value pairs:

            
        let person = {
            name: "John",
            age: 30,
            greet: function() {
                return "Hello, " + this.name;
            }
        };
        console.log(person.greet()); // Hello, John
            

        


        

            
17. JS Object Properties

            
Access and modify object properties using dot notation or bracket notation:

            
        let car = { brand: "Toyota", model: "Corolla" };
        console.log(car.brand); // Toyota
        car.year = 2020;
            

        


        

            
18. JS Object Methods

            
Methods are functions that are properties of an object:

            
        let person = {
            name: "Alice",
            greet: function() {
                return "Hello, " + this.name;
            }
        };
        console.log(person.greet()); // Hello, Alice
            

        


        

            
19. JS Object Display

            
Display object properties and methods:

            
        let car = { brand: "Ford", model: "Mustang" };
        console.log(car); // { brand: "Ford", model: "Mustang" }
            

        


        

            
20. JS Object Constructors

            
Use constructors to create new objects using the new keyword:

            
        function Person(name, age) {
            this.name = name;
            this.age = age;
        }
        let john = new Person("John", 25);
        console.log(john.name); // John
            

        


        

            
21. JS Events

            
Events are actions that JavaScript can respond to, such as user interactions:

            
        document.getElementById("myButton").addEventListener("click", function() {
            alert("Button clicked!");
        });
            

        


        

            
22. JS Strings

            
Strings are sequences of characters enclosed in quotes:

            
        let str = "Hello, World!";
        console.log(str); // Hello, World!
            

        


        

            
23. JS String Methods

            
Strings have various methods like toUpperCase() and slice():

            
        let str = "JavaScript";
        console.log(str.toUpperCase()); // JAVASCRIPT
        console.log(str.slice(0, 4)); // Java
            

        


        

            
24. JS String Search

            
Search for substrings within a string using indexOf():

            
        let str = "Hello, World!";
        console.log(str.indexOf("World")); // 7
            

        


        

            
25. JS String Templates

            
Template literals allow embedded expressions and multi-line strings:

            
        let name = "John";
        let message = `Hello, ${name}!`;
        console.log(message); // Hello, John!
            

        


        

            
26. JS Numbers

            
Numbers represent both integer and floating-point values:

            
        let int = 42;
        let float = 3.14;
        console.log(int); // 42
        console.log(float); // 3.14
            

        


        

            
27. JS BigInt

            
BigInt allows representation of large integers beyond the limits of the Number type:

            
        let bigInt = 1234567890123456789012345678901234567890n;
        console.log(bigInt); // 1234567890123456789012345678901234567890n
            

        


        

            
28. JS Number Methods

            
Number methods include toFixed() and toString():

            
        let num = 10.567;
        console.log(num.toFixed(2)); // 10.57
        console.log(num.toString()); // "10.567"
            

        


        

            
29. JS Number Properties

            
Number properties include constants like Number.MAX_VALUE:

            
        console.log(Number.MAX_VALUE); // 1.7976931348623157e+308
            

        


        

            
30. JS Arrays

            
Arrays store lists of values:

            
        let fruits = ["Apple", "Banana", "Cherry"];
        console.log(fruits); // ["Apple", "Banana", "Cherry"]
            

        


        

            
31. JS Array Methods

            
Array methods include push(), pop(), and splice():

            
        let numbers = [1, 2, 3];
        numbers.push(4); // [1, 2, 3, 4]
        numbers.pop(); // [1, 2, 3]
        numbers.splice(1, 1, 10); // [1, 10, 3]
            

        


        

            
32. JS Array Search

            
Search arrays using indexOf() and find():

            
        let arr = [10, 20, 30];
        console.log(arr.indexOf(20)); // 1
        console.log(arr.find(x => x > 15)); // 20
            

        


        

            
33. JS Array Sort

            
Sort arrays using sort():

            
        let numbers = [4, 2, 5, 1, 3];
        numbers.sort();
        console.log(numbers); // [1, 2, 3, 4, 5]
            

        


        

            
34. JS Array Iteration

            
Iterate over arrays using forEach():

            
        let numbers = [1, 2, 3];
        numbers.forEach(num => console.log(num));
            

        


        

            
35. JS Array Const

            
Arrays declared with const cannot be reassigned, but their contents can be modified:

            
        const fruits = ["Apple", "Banana"];
        fruits.push("Cherry");
        console.log(fruits); // ["Apple", "Banana", "Cherry"]
            

        


        

            
36. JS Dates

            
Use the Date object to work with dates and times:

            
        let today = new Date();
        console.log(today); // Current date and time
            

        


        

            
37. JS Date Formats

            
Format dates using methods like toLocaleDateString():

            
        let date = new Date();
        console.log(date.toLocaleDateString()); // e.g., 7/21/2024
            

        


        

            
38. JS Date Get Methods

            
Get specific date and time components using methods like getFullYear():

            
        let date = new Date();
        console.log(date.getFullYear()); // e.g., 2024
            

        


        

            
39. JS Date Set Methods

            
Set specific date and time components using methods like setFullYear():

            
        let date = new Date();
        date.setFullYear(2025);
        console.log(date.getFullYear()); // 2025
            

        


        

            
40. JS Math

            
The Math object provides mathematical constants and functions:

            
        console.log(Math.PI); // 3.141592653589793
        console.log(Math.sqrt(16)); // 4
            

        


        

            
41. JS Random

            
Generate random numbers using Math.random():

            
        console.log(Math.random()); // Random number between 0 and 1
            

        


        

            
42. JS Booleans

            
Booleans represent true or false values:

            
        let isTrue = true;
        let isFalse = false;
        console.log(isTrue); // true
            

        


        

            
43. JS Comparisons

            
Comparison operators include ==, ===, !=, and >:

            
        let a = 5;
        let b = 10;
        console.log(a == b); // false
        console.log(a < b); // true
            

        


        

            
44. JS If Else

            
Control flow using if...else statements:

            
        let age = 18;
        if (age >= 18) {
            console.log("Adult");
        } else {
            console.log("Minor");
        }
            

        


        

            
45. JS Switch

            
Use switch for multiple conditional branches:

            
        let day = 2;
        switch (day) {
            case 1:
                console.log("Monday");
                break;
            case 2:
                console.log("Tuesday");
                break;
            default:
                console.log("Another day");
        }
            

        


        

            
46. JS Loop For

            
Loop through code blocks using for loops:

            
        for (let i = 0; i < 5; i++) {
            console.log(i);
        }
            

        


        

            
47. JS Loop For In

            
Loop through object properties using for...in:

            
        let person = { name: "Alice", age: 25 };
        for (let key in person) {
            console.log(key + ": " + person[key]);
        }
            

        


        

            
48. JS Loop For Of

            
Loop through iterable objects using for...of:

            
        let numbers = [1, 2, 3];
        for (let num of numbers) {
            console.log(num);
        }
            

        


        

            
49. JS Loop While

            
Use while loops for repeated actions until a condition is false:

            
        let i = 0;
        while (i < 5) {
            console.log(i);
            i++;
        }
            

        


        

            
50. JS Break

            
Exit loops prematurely using break:

            
        for (let i = 0; i < 10; i++) {
            if (i === 5) break;
            console.log(i);
        }
            

        


        

            
51. JS Iterables

            
Iterables are objects that can be iterated over using loops. Arrays and strings are examples:

            
        let str = "hello";
        for (let char of str) {
            console.log(char);
        }
            

        


        

            
52. JS Sets

            
Sets store unique values. Use methods like add() and has():

            
        let mySet = new Set();
        mySet.add(1);
        mySet.add(2);
        console.log(mySet.has(1)); // true
            

        


        

            
53. JS Set Methods

            
Set methods include delete() and clear():

            
        let mySet = new Set([1, 2, 3]);
        mySet.delete(2);
        console.log(mySet); // Set {1, 3}
            

        


        

            
54. JS Maps

            
Maps store key-value pairs and maintain the insertion order. Use methods like set() and get():

            
        let myMap = new Map();
        myMap.set("name", "Alice");
        console.log(myMap.get("name")); // Alice
            

        


        

            
55. JS Map Methods

            
Map methods include delete() and has():

            
        let myMap = new Map([["key1", "value1"], ["key2", "value2"]]);
        myMap.delete("key1");
        console.log(myMap); // Map { "key2" => "value2" }
            

        


        

            
56. JS Typeof

            
Use typeof to determine the type of a variable:

            
        let num = 42;
        console.log(typeof num); // number
            

        


        

            
57. JS Type Conversion

            
Convert between types using functions like String() and Number():

            
        let str = "123";
        let num = Number(str);
        console.log(num); // 123
            

        


        

            
58. JS Destructuring

            
Destructure arrays and objects to extract values:

            
        let [a, b] = [1, 2];
        let { name, age } = { name: "Alice", age: 25 };
        console.log(a); // 1
        console.log(name); // Alice
            

        


        

            
59. JS Bitwise

            
Bitwise operators perform operations on binary representations of numbers:

            
        let x = 5; // 0101 in binary
        let y = 3; // 0011 in binary
        console.log(x & y); // 1 (0001 in binary)
            

        


        

            
60. JS RegExp

            
Regular expressions are patterns used for matching character combinations in strings:

            
        let regex = /hello/i;
        console.log(regex.test("Hello World")); // true
            

        


        

            
61. JS Precedence

            
Operator precedence determines the order in which operations are performed:

            
        let result = 2 + 3 * 4;
        console.log(result); // 14 (multiplication has higher precedence)
            

        


        

            
62. JS Errors

            
Handle errors using try...catch:

            
        try {
            let result = 10 / 0;
        } catch (error) {
            console.log("Error: " + error.message);
        }
            

        


        

            
63. JS Scope

            
Scope refers to the visibility of variables. Variables declared with let and const have block scope:

            
        {
            let x = 10;
        }
        console.log(x); // Error: x is not defined
            

        


        

            
64. JS Hoisting

            
Hoisting allows variables and function declarations to be used before they are declared:

            
        console.log(x); // undefined
        var x = 5;
            

        


        

            
65. JS Strict Mode

            
Enable strict mode by adding "use strict"; at the beginning of a script or function:

            
        "use strict";
        let x = 10;
        // x = undeclaredVar; // Error: undeclaredVar is not defined
            

        


        

            
66. JS this Keyword

            
The this keyword refers to the current object:

            
        let person = {
            name: "Alice",
            greet: function() {
                return "Hello, " + this.name;
            }
        };
        console.log(person.greet()); // Hello, Alice
            

        


        

            
67. JS Arrow Functions

            
Arrow functions provide a concise syntax for function expressions:

            
        let add = (a, b) => a + b;
        console.log(add(2, 3)); // 5
            

        


        

            
68. JS Classes

            
Classes are templates for creating objects with shared properties and methods:

            
        class Person {
            constructor(name, age) {
                this.name = name;
                this.age = age;
            }
            greet() {
                return "Hello, " + this.name;
            }
        }
        let john = new Person("John", 30);
        console.log(john.greet()); // Hello, John
            

        


        

            
69. JS Modules

            
Modules allow you to import and export code between files:

            
        // math.js
        export function add(a, b) {
            return a + b;
        }
        // main.js
        import { add } from './math.js';
        console.log(add(2, 3)); // 5
            

        


        

            
70. JS JSON

            
JSON (JavaScript Object Notation) is a lightweight data-interchange format:

            
        let obj = { name: "Alice", age: 30 };
        let json = JSON.stringify(obj);
        console.log(json); // {"name":"Alice","age":30}
            

        


        

            
71. JS Debugging

            
Use the debugger statement to pause execution and inspect values:

            
        function add(a, b) {
            debugger;
            return a + b;
        }
        console.log(add(2, 3)); // 5
            

        


        

            
72. JS Style Guide

            
Follow coding conventions for readability and maintainability:

            
        let firstName = "Alice"; // Camel case for variables
        function sayHello(name) {
            return `Hello, ${name}`;
        }
            

        


        

            
73. JS Best Practices

            
Adopt best practices for writing clean and efficient code:

            
        function sum(a, b) {
            return a + b;
        }
        console.log(sum(5, 10)); // 15
            

        


        

            
74. JS Mistakes

            
Avoid common mistakes such as omitting return statements or using undeclared variables:

            
        function multiply(a, b) {
            return a * b; // Ensure return statement is present
        }
        console.log(multiply(2, 3)); // 6
            

        


        

            
75. JS Performance

            
Optimize code for better performance, such as avoiding unnecessary computations:

            
        let numbers = [1, 2, 3, 4, 5];
        let sum = numbers.reduce((total, num) => total + num, 0);
        console.log(sum); // 15
            

        


        

            
76. JS Reserved Words

            
Reserved words cannot be used as identifiers. Examples include class, return, and var:

            
        let class = "example"; // Error: Unexpected strict mode reserved word
