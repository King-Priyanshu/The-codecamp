Python HOME

Welcome to Python programming! Python is a popular programming language known for its simplicity and readability.

Python Get Started

To get started with Python, you need to install it on your computer. Visit the official Python Downloads page to download and install Python.

Python Intro

Python is a high-level, interpreted programming language with dynamic semantics. It is used for web development, data analysis, artificial intelligence, and more.

Python Syntax

Python syntax is clean and easy to understand. Here is a basic example:

                
                    # This is a comment
                    print("Hello, World!")

Python Comments

Comments are used to explain code. Single-line comments start with a #:

                
                    # This is a single-line comment

Python Variables

Variables are used to store data. You do not need to declare variables before using them:

                
                    x = 5
                    y = "Hello, World!"

Python Data Types

Python supports various data types such as integers, floats, strings, and booleans...

                
                    x = 10      # Integer
                    y = 2.5     # Float
                    z = "Text"  # String
                    w = True    # Boolean

Python Numbers

Python has three numerical types: int, float, and complex:

                
                    x = 1      # int
                    y = 2.8    # float
                    z = 1j     # complex

Python Casting

You can convert between data types using casting:

                
                    x = int(2.8)    # Convert float to int
                    y = float(3)    # Convert int to float
                    z = str(10)     # Convert int to string

Python Strings

Strings are arrays of bytes representing Unicode characters:

                
                    text = "Hello, World!"
                    print(text[0])   # H
                    print(text[2:5]) # llo

Python Booleans

Booleans represent one of two values: True or False:

                
                    x = True
                    y = False

Python Operators

Python supports various operators such as arithmetic, comparison, and logical operators:

                
                    x = 10 + 5   # Arithmetic
                    y = 10 > 5   # Comparison
                    z = True and False  # Logical

Python Lists

Lists are used to store multiple items in a single variable:

                
                    my_list = ["apple", "banana", "cherry"]
                    print(my_list[1])  # banana

Python Tuples

Tuples are similar to lists but are immutable:

                
                    my_tuple = ("apple", "banana", "cherry")
                    print(my_tuple[1])  # banana

Python Sets

Sets are unordered collections of unique items:

                
                    my_set = {"apple", "banana", "cherry"}
                    print("apple" in my_set)  # True

Python Dictionaries

Dictionaries store data in key-value pairs:

                
                    my_dict = {"name": "Alice", "age": 25}
                    print(my_dict["name"])  # Alice

Python If...Else

Conditional statements are used to perform different actions based on conditions:

                
                    if x > y:
                        print("x is greater than y")
                    else:
                        print("x is not greater than y")

Python While Loops

While loops are used to execute a block of code as long as a condition is true:

                
                    i = 1
                    while i < 6:
                        print(i)
                        i += 1

Python For Loops

For loops are used to iterate over a sequence (list, tuple, dictionary, set, or string):

                
                    for x in my_list:
                        print(x)

Python Functions

Functions are blocks of code that only run when called:

                
                    def my_function():
                        print("Hello from a function")

                    my_function()

Python Lambda

Lambda functions are small anonymous functions:

                
                    x = lambda a, b: a + b
                    print(x(5, 6))  # 11

Python Arrays

Arrays are used to store multiple values in a single variable:

                
                    import array as arr
                    my_array = arr.array('i', [1, 2, 3, 4])
                    print(my_array[1])  # 2

Python Classes/Objects

Python is an object-oriented programming language:

                
                    class MyClass:
                        def __init__(self, name):
                            self.name = name

                        def greet(self):
                            print(f"Hello, {self.name}")

                    obj = MyClass("Alice")
                    obj.greet()  # Hello, Alice

Python Inheritance

Inheritance allows us to define a class that inherits all the methods and properties from another class:

                
                    class Parent:
                        def __init__(self, name):
                            self.name = name

                    class Child(Parent):
                        def greet(self):
                            print(f"Hello, {self.name}")

                    obj = Child("Alice")
                    obj.greet()  # Hello, Alice

Python Iterators

Iterators are used to iterate over a sequence of elements:

                
                    my_list = [1, 2, 3]
                    my_iter = iter(my_list)
                    print(next(my_iter))  # 1

Python Polymorphism

Polymorphism allows us to define methods in the child class with the same name as defined in their parent class:

                
                    class Animal:
                        def sound(self):
                            pass

                    class Dog(Animal):
                        def sound(self):
                            return "Woof"

                    class Cat(Animal):
                        def sound(self):
                            return "Meow"

                    def make_sound(animal):
                        print(animal.sound())

                    make_sound(Dog())  # Woof
                    make_sound(Cat())  # Meow

Python Scope

Scope determines the visibility of a variable within a block of code:

                
                    def my_function():
                        x = 10  # Local scope

                    x = 20  # Global scope
                    my_function()
                    print(x)  # 20

Python Modules

Modules are files containing Python code that can be imported into other Python files:

                
                    # mymodule.py
                    def greeting(name):
                        print(f"Hello, {name}")

                    # main.py
                    import mymodule
                    mymodule.greeting("Alice")  # Hello, Alice

Python Dates

Python has a built-in module called datetime to work with dates and times:

                
                    import datetime
                    x = datetime.datetime.now()
                    print(x)  # 2024-07-20 12:34:56.789123

Python Math

Python has a built-in module called math that provides mathematical functions:

                
                    import math
                    x = math.sqrt(16)
                    print(x)  # 4.0

Python JSON

Python has a built-in module called json to work with JSON data:

                
                    import json
                    x = '{"name": "Alice", "age": 25}'
                    y = json.loads(x)
                    print(y["name"])  # Alice

Python RegEx

Python has a built-in module called re to work with regular expressions:

                
                    import re
                    txt = "The rain in Spain"
                    x = re.search("ai", txt)
                    print(x)  #

Python PIP

PIP is a package manager for Python packages. You can install packages using the command:

                
                    pip install package_name

Python Try...Except

Try...except blocks are used to handle exceptions:

                
                    try:
                        x = 10 / 0
                    except ZeroDivisionError:
                        print("Cannot divide by zero")

Python User Input

You can get user input using the input function:

                
                    name = input("Enter your name: ")
                    print(f"Hello, {name}")

Python String Formatting

String formatting allows you to format strings in various ways:

                
                    age = 25
                    txt = f"My name is Alice, and I am {age} years old."
                    print(txt)  # My name is Alice, and I am 25 years old.
