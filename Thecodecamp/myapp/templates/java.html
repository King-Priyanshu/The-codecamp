Java Intro

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is intended to let developers "write once, run anywhere" (WORA), meaning that code that runs on one platform does not need to be recompiled to run on another.

Java Get Started

To get started with Java, you need to install the Java Development Kit (JDK) and set up your development environment. You can write Java code using a simple text editor or an Integrated Development Environment (IDE) like Eclipse or IntelliJ IDEA. Here’s a basic example of a Java program: 

        
            public class HelloWorld {
                public static void main(String[] args) {
                    System.out.println("Hello, World!");
                }
            }

Java Syntax

Java syntax is the set of rules defining how a Java program is written and interpreted. Java syntax is similar to C and C++. Here’s an example: 

        
            int a = 10;
            int b = 20;
            int sum = a + b;
            System.out.println("Sum: " + sum);

Java Output

Output in Java is often displayed using the System.out.println() method. This method prints text to the console. Example: 

        
            System.out.println("This is Java output.");

Java Comments

Comments in Java are used to explain the code and are ignored by the compiler. They can be single-line or multi-line. Example: 

        
            // This is a single-line comment
            /* This is a
               multi-line comment */

Java Variables

Variables in Java are containers for storing data values. Each variable must be declared with a data type. Example: 

        
            int age = 30;
            String name = "John";

Java Data Types

Java has two categories of data types: primitive and reference types. Primitive types include int, char, boolean, etc. Example: 

        
            int number = 100;
            double decimal = 10.5;
            boolean isTrue = true;

Java Type Casting

Type casting is used to convert a variable from one data type to another. There are two types: implicit and explicit. Example: 

       
    // Implicit Casting (Widening)
    int num = 10;
    double doubleNum = num; // int to double

    // Explicit Casting (Narrowing)
    double pi = 3.14;
    int intPi = (int) pi; // double to int

    // Example with characters
    char letter = 'A';
    int asciiValue = letter; // implicit casting from char to int

    // Example with large to small data types
    long largeNumber = 100000L;
    int smallerNumber = (int) largeNumber; // explicit casting from long to int

    // Example with potential data loss
    float preciseNumber = 10.99f;
    int roundedNumber = (int) preciseNumber; // fractional part is lost

Java Operators

Operators in Java are used to perform operations on variables and values. Common operators include arithmetic, relational, and logical operators. Example: 

        
            int a = 5;
            int b = 10;
            int sum = a + b; // Arithmetic operator
            boolean isEqual = (a == b); // Relational operator
            boolean result = (a < b) && (b > 0); // Logical operator

Java Strings

Strings in Java are objects that represent a sequence of characters. You can create a string using double quotes. Example: 

        
            String greeting = "Hello, World!";
            System.out.println(greeting);

Java Math

Java provides a Math class that contains methods for performing basic numeric operations such as exponentiation, logarithms, square roots, and trigonometric functions. Example: 

        
            double squareRoot = Math.sqrt(25); // 5.0
            double power = Math.pow(2, 3); // 8.0
            System.out.println("Square root: " + squareRoot);
            System.out.println("Power: " + power);

Java Booleans

Booleans in Java represent one of two values: true or false. They are used for conditional statements and logic operations. Example: 

        
            boolean isJavaFun = true;
            boolean isFishTasty = false;
            System.out.println("Is Java fun? " + isJavaFun);
            System.out.println("Is fish tasty? " + isFishTasty);

Java If...Else

The if...else statement executes a block of code if its condition is true and another block if it is false. Example: 

        
            int number = 10;
            if (number > 0) {
                System.out.println("Number is positive.");
            } else {
                System.out.println("Number is not positive.");
            }

Java Switch

The switch statement executes one block of code among many options based on the value of an expression. Example: 

        
            int day = 3;
            switch (day) {
                case 1:
                    System.out.println("Monday");
                    break;
                case 2:
                    System.out.println("Tuesday");
                    break;
                case 3:
                    System.out.println("Wednesday");
                    break;
                default:
                    System.out.println("Invalid day");
            }

Java While Loop

The while loop repeatedly executes a block of code as long as its condition is true. Example: 

        
            int i = 0;
            while (i < 5) {
                System.out.println(i);
                i++;
            }

Java For Loop

The for loop is used to iterate over a range of values. It is often used when the number of iterations is known. Example: 

        
            for (int j = 0; j < 5; j++) {
                System.out.println(j);
            }

Java Break/Continue

The break statement exits from a loop or switch statement, and the continue statement skips the current iteration of a loop and proceeds with the next iteration. Example: 

        
            for (int k = 0; k < 10; k++) {
                if (k == 5) {
                    break; // Exits the loop when k is 5
                }
                if (k % 2 == 0) {
                    continue; // Skips the rest of the loop when k is even
                }
                System.out.println(k);
            }

Java Arrays

Arrays in Java are used to store multiple values of the same type in a single variable. Example: 

        
            int[] numbers = {1, 2, 3, 4, 5};
            for (int n : numbers) {
                System.out.println(n);
            }

Java OOP Concepts

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which contain data and methods to manipulate that data. The four main principles of OOP are:

  • Encapsulation: Bundling the data (variables) and methods (functions) that operate on the data into a single unit called a class.
  • Inheritance: Mechanism where one class can inherit fields and methods from another.
  • Polymorphism: The ability of an object to take on many forms, typically through method overriding or overloading.
  • Abstraction: Hiding the implementation details and showing only the essential features of the object.

Java Classes and Objects

A class is a blueprint for creating objects (instances). A object is an instance of a class. Here’s an example: 

    
        class Car {
            String model;
            int year;

            // Constructor
            Car(String model, int year) {
                this.model = model;
                this.year = year;
            }

            // Method
            void displayInfo() {
                System.out.println("Model: " + model + ", Year: " + year);
            }
        }

        // Create an object
        Car car1 = new Car("Tesla Model 3", 2023);
        car1.displayInfo();

Java Constructors

A constructor is a special method used to initialize objects. It is called when an object of a class is created. Java provides two types of constructors: default constructors and parameterized constructors. 

    
        class Car {
            String model;
            int year;

            // Default constructor
            Car() {
                model = "Unknown";
                year = 0;
            }

            // Parameterized constructor
            Car(String model, int year) {
                this.model = model;
                this.year = year;
            }
        }

        // Creating objects
        Car car1 = new Car();
        Car car2 = new Car("Tesla", 2023);

Java Inheritance

Inheritance allows one class to inherit fields and methods from another class. In Java, inheritance is achieved by using the extends keyword. 

    
        class Animal {
            void sound() {
                System.out.println("Animal makes a sound");
            }
        }

        class Dog extends Animal {
            void sound() {
                System.out.println("Dog barks");
            }
        }

        Dog dog = new Dog();
        dog.sound(); // Outputs: Dog barks

Java Polymorphism

Polymorphism allows a method or object to take multiple forms. In Java, it is achieved through method overriding and method overloading. 

    
        // Method Overriding
        class Animal {
            void sound() {
                System.out.println("Animal makes a sound");
            }
        }

        class Dog extends Animal {
            @Override
            void sound() {
                System.out.println("Dog barks");
            }
        }

        Animal animal = new Dog();
        animal.sound(); // Outputs: Dog barks

Java Abstraction

Abstraction is the concept of hiding implementation details and showing only the essential features of the object. Java achieves abstraction through abstract classes and interfaces. 

    
        abstract class Animal {
            abstract void sound();
        }

        class Dog extends Animal {
            void sound() {
                System.out.println("Dog barks");
            }
        }

        Animal animal = new Dog();
        animal.sound(); // Outputs: Dog barks

Java Encapsulation

Encapsulation is the concept of wrapping data (variables) and code (methods) together as a single unit. It is achieved using access modifiers like private, protected, and public. 

    
        class Person {
            private String name;
            private int age;

            // Getter and Setter methods
            public String getName() {
                return name;
            }

            public void setName(String name) {
                this.name = name;
            }

            public int getAge() {
                return age;
            }

            public void setAge(int age) {
                this.age = age;
            }
        }

        Person person = new Person();
        person.setName("John");
        person.setAge(30);
        System.out.println(person.getName() + " is " + person.getAge() + " years old.");

Java Interfaces

An interface in Java is a reference type, similar to a class, that can contain only constants, method signatures, default methods, static methods, and nested types. It is used to represent a contract for what a class can do, without specifying how. 

    
        interface Animal {
            void sound();
        }

        class Dog implements Animal {
            public void sound() {
                System.out.println("Dog barks");
            }
        }

        Dog dog = new Dog();
        dog.sound(); // Outputs: Dog barks

Java Packages

A package in Java is a namespace that groups related classes and interfaces. Packages are used to avoid name conflicts and to organize the code in a modular way. Java provides two types of packages: built-in and user-defined. 

    
        // Importing a built-in package
        import java.util.Scanner;

        public class MyClass {
            public static void main(String[] args) {
                Scanner scanner = new Scanner(System.in);
                System.out.println("Enter your name: ");
                String name = scanner.nextLine();
                System.out.println("Hello, " + name);
            }
        }

Java Exceptions

Exceptions are unwanted or unexpected events that disrupt the normal flow of a program. Java provides a robust mechanism for handling exceptions through try, catch, and finally blocks. 

    
        try {
            int[] numbers = {1, 2, 3};
            System.out.println(numbers[5]); // This will cause an exception
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("Error: " + e.getMessage());
        } finally {
            System.out.println("This block always executes.");
        }

Java Multithreading

Multithreading is a process of executing multiple threads simultaneously. It allows Java programs to perform multiple tasks concurrently. Java provides the Thread class and the Runnable interface to create threads. 

    
        class MyThread extends Thread {
            public void run() {
                System.out.println("Thread is running.");
            }
        }

        public class Main {
            public static void main(String[] args) {
                MyThread t = new MyThread();
                t.start(); // Start the thread
            }
        }

Java File Handling

Java provides the java.io package for file handling. It includes classes like File, FileReader, and FileWriter for reading from and writing to files. 

    
        import java.io.*;

        public class FileExample {
            public static void main(String[] args) throws IOException {
                FileWriter writer = new FileWriter("example.txt");
                writer.write("Hello, world!");
                writer.close();

                BufferedReader reader = new BufferedReader(new FileReader("example.txt"));
                String line = reader.readLine();
                System.out.println(line); // Outputs: Hello, world!
                reader.close();
            }
        }

Java Generics

Generics allow classes, interfaces, and methods to operate on objects of various types while providing compile-time type safety. This prevents runtime errors due to type mismatches. 

    
        class Box {
            private T value;

            public void setValue(T value) {
                this.value = value;
            }

            public T getValue() {
                return value;
            }
        }

        public class Main {
            public static void main(String[] args) {
                Box intBox = new Box<>();
                intBox.setValue(10);
                System.out.println(intBox.getValue()); // Outputs: 10
            }
        }

Java Collections

The java.util package provides a set of classes and interfaces that implement commonly used collections like lists, sets, and maps. These collections help store and manipulate data efficiently. 

    
        import java.util.*;

        public class CollectionExample {
            public static void main(String[] args) {
                List fruits = new ArrayList<>();
                fruits.add("Apple");
                fruits.add("Banana");
                fruits.add("Orange");

                for (String fruit : fruits) {
                    System.out.println(fruit);
                }
            }
        }

Java Streams

Java Streams API allows functional-style operations on streams of elements. It enables processing sequences of elements in a declarative way using methods like filter, map, and reduce. 

    
        import java.util.*;
        import java.util.stream.*;

        public class StreamExample {
            public static void main(String[] args) {
                List numbers = Arrays.asList(1, 2, 3, 4, 5);
                int sum = numbers.stream()
                                 .filter(n -> n % 2 == 0)
                                 .mapToInt(Integer::intValue)
                                 .sum();
                System.out.println("Sum of even numbers: " + sum); // Outputs: 6
            }
        }

Java Annotations

Annotations provide metadata about the program but are not part of the program's logic. They are used to give instructions to the compiler or tools, like marking a method as deprecated or indicating that a method should override another method. 

    
        @Override
        public String toString() {
            return "Custom toString method";
        }

        @Deprecated
        public void oldMethod() {
            System.out.println("This method is deprecated.");
        }

Java Lambda Expressions

Lambda expressions allow you to pass behavior as a parameter to a method, simplifying the code. They are often used to implement functional interfaces. 

    
        interface MathOperation {
            int operation(int a, int b);
        }

        public class Main {
            public static void main(String[] args) {
                MathOperation add = (a, b) -> a + b;
                System.out.println(add.operation(5, 3)); // Outputs: 8
            }
        }

Java Date and Time

The java.time package in Java provides a comprehensive and standardized way of handling date and time. It includes classes such as LocalDate, LocalTime, and LocalDateTime. 

    
        import java.time.*;

        public class DateTimeExample {
            public static void main(String[] args) {
                LocalDate today = LocalDate.now();
                System.out.println("Today's date: " + today); // Outputs: Current date

                LocalTime time = LocalTime.now();
                System.out.println("Current time: " + time); // Outputs: Current time
            }
        }
