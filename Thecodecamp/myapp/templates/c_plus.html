C++ Introduction

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It is widely used for system/software development, game development, drivers, client-server applications, and embedded firmware.

Get Started

To get started with C++, you need to set up an environment that allows you to write and execute C++ code. You can use an Integrated Development Environment (IDE) like Visual Studio, Code::Blocks, or a simple text editor like Visual Studio Code combined with a compiler like GCC (GNU Compiler Collection).

C++ Syntax

C++ syntax is quite similar to C but with additional features. Here's a simple example of a C++ program:


            #include <iostream>  // Preprocessor directive for input-output stream

            int main() {
                std::cout << "Hello, World!" << std::endl;  // Print Hello, World!
                return 0;  // Return 0 to indicate that the program ended successfully
            }

Output

To output text in C++, you use the std::cout object along with the insertion operator <<.


            #include <iostream>

            int main() {
                std::cout << "Hello, World!" << std::endl;
                return 0;
            }

Comments

Comments are used to explain code and make it more readable. There are two types of comments in C++:

  • Single-line comments: Start with //
  • Multi-line comments: Enclosed in /* ... */

            #include <iostream>

            int main() {
                // This is a single-line comment
                std::cout << "Hello, World!" << std::endl; /* This is a multi-line comment */
                return 0;
            }

Variables

Variables are containers for storing data values. In C++, you must declare a variable before you can use it.


            #include <iostream>

            int main() {
                int myNum = 15;  // Integer (whole number)
                double myFloatNum = 5.99;  // Floating point number
                char myLetter = 'D';  // Character
                std::string myText = "Hello";  // String (text)
                bool myBoolean = true;  // Boolean (true/false)
                std::cout << myNum << std::endl;
                std::cout << myFloatNum << std::endl;
                std::cout << myLetter << std::endl;
                std::cout << myText << std::endl;
                std::cout << myBoolean << std::endl;
                return 0;
            }

User Input

You can use the std::cin object to get user input in C++.


            #include <iostream>

            int main() {
                int myNum;
                std::cout << "Enter a number: ";
                std::cin >> myNum;
                std::cout << "You entered: " << myNum << std::endl;
                return 0;
            }

Data Types

C++ provides several data types for different kinds of variables.

  • int: Integer
  • float: Floating point number
  • double: Double-precision floating point number

  • char: Character
  • bool: Boolean
  • string: String (text)

Operators

Operators are used to perform operations on variables and values. Examples include:

  • Arithmetic operators: +, -, *, /, %

  • Comparison operators: ==, !=, >, <, >=, <=

  • Logical operators: &&, ||, !

            #include <iostream>

            int main() {
                int x = 5;
                int y = 10;
                std::cout << (x + y) << std::endl;  // Addition
                std::cout << (x == y) << std::endl;  // Comparison
                std::cout << (x < y && x > 0) << std::endl;  // Logical AND
                return 0;
            }

Strings

Strings in C++ are used to store text. The standard library provides a string class for handling strings.


            #include <iostream>
            #include <string>

            int main() {
                std::string greeting = "Hello";
                std::cout << greeting << std::endl;
                return 0;
            }

Math

C++ provides a <cmath> library for common mathematical functions.


            #include <iostream>
            #include <cmath>

            int main() {
                std::cout << std::sqrt(16) << std::endl;  // Square root
                std::cout << std::pow(2, 3) << std::endl;  // Power
                return 0;
            }

Booleans

Booleans represent two values: true and false.


            #include <iostream>

            int main() {
                bool isCodingFun = true;
                bool isFishTasty = false;
                std::cout << isCodingFun << std::endl;  // Outputs 1 (true)
                std::cout << isFishTasty << std::endl;  // Outputs 0 (false)
                return 0;
            }

Conditions

Conditional statements are used to perform different actions based on different conditions. The if statement is the most basic conditional statement in C++.


            #include <iostream>

            int main() {
                if (5 > 2) {
                    std::cout << "5 is greater than 2." << std::endl;
                }
                return 0;
            }

Switch

The switch statement allows a variable to be tested for equality against a list of values.


            #include <iostream>

            int main() {
                int day = 4;
                switch (day) {
                    case 1:
                        std::cout << "Monday" << std::endl;
                        break;
                    case 2:
                        std::cout << "Tuesday" << std::endl;
                        break;
                    case 3:
                        std::cout << "Wednesday" << std::endl;
                        break;
                    case 4:
                        std::cout << "Thursday" << std::endl;
                        break;
                    case 5:
                        std::cout << "Friday" << std::endl;
                        break;
                    case 6:
                        std::cout << "Saturday" << std::endl;
                        break;
                    case 7:
                        std::cout << "Sunday" << std::endl;
                        break;
                    default:
                        std::cout << "Invalid day" << std::endl;
                }
                return 0;
            }

While Loop

A while loop repeats a statement or group of statements while a given condition is true. It tests the condition before executing the loop body.


            #include <iostream>

            int main() {
                int i = 0;
                while (i < 5) {
                    std::cout << i << std::endl;
                    i++;
                }
                return 0;
            }

For Loop

A for loop is a more compact version of the while loop. It includes the initialization, condition, and increment/decrement in one line.


            #include <iostream>

            int main() {
                for (int i = 0; i < 5; i++) {
                    std::cout << i << std::endl;
                }
                return 0;
            }

Break and Continue

The break statement terminates the loop, and the continue statement skips the current iteration and moves to the next one.


            #include <iostream>

            int main() {
                for (int i = 0; i < 10; i++) {
                    if (i == 4) {
                        break;  // Terminate the loop if i is 4
                    }
                    if (i == 2) {
                        continue;  // Skip the iteration if i is 2
                    }
                    std::cout << i << std::endl;
                }
                return 0;
            }

Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.


            #include <iostream>

            int main() {
                int numbers[] = {1, 2, 3, 4, 5};
                for (int i = 0; i < 5; i++) {
                    std::cout << numbers[i] << std::endl;
                }
                return 0;
            }

Structures

Structures (structs) are used to group variables of different data types under a single name.


            #include <iostream>
            #include <string>

            struct Person {
                std::string name;
                int age;
            };

            int main() {
                Person person1;
                person1.name = "John";
                person1.age = 30;
                std::cout << person1.name << " is " << person1.age << " years old." << std::endl;
                return 0;
            }

Enums

Enums are user-defined types that consist of a set of named integral constants, called enumerators.


            #include <iostream>

            enum Color {RED, GREEN, BLUE};

            int main() {
                Color myColor = GREEN;
                if (myColor == GREEN) {
                    std::cout << "My favorite color is green." << std::endl;
                }
                return 0;
            }

References

References are an alternative to pointers that provide a way to refer to the same variable by another name.


            #include <iostream>

            int main() {
                int a = 10;
                int &ref = a;
                ref++;
                std::cout << "a: " << a << std::endl;  // Outputs 11
                std::cout << "ref: " << ref << std::endl;  // Outputs 11
                return 0;
            }

Pointers

Pointers are variables that store memory addresses of other variables.


            #include <iostream>

            int main() {
                int var = 20;
                int *ptr = &var;
                std::cout << "Value of var: " << var << std::endl;
                std::cout << "Address of var: " << ptr << std::endl;
                std::cout << "Value pointed by ptr: " << *ptr << std::endl;
                return 0;
            }
